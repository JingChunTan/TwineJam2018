See 'Gauntlet' template for more info on using RPG-style mechanics. This structure is best suited for digital fiction, where state-tracking variables can mean early decisions affect later choices. Obviously, this isn't easily doable on paper, but some clever thinking can still allow this structure to work. The most important part of this structure is the way choices direct the player to many different passages, then return them to a central path. In a way, it's like a more forgiving gauntlet: with more choices and less risk to the player for exploring them. Resist the urge to change too many of the branch nodes into endings if you want to retain this style, as doing so will simply turn into a Gauntlet.
